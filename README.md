# Dual-image-cutter
A Gradio application that can crop two images at the same position at the same time
Your file should be placed as follows

-input
--image1_1.png
--image1_2.png
--image2_1.png
--image2_2.png
--image3_1.png
--image3_2.png
--...

"image1_1.png" and "image1_2.png" will be processed at the same time.And so on.
